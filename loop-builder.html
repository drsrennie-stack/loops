Loop Builder

Make a new loop, or add cards to a loop you already posted. It runs in your browser and downloads a ready ZIP.

Nothing is uploaded anywhere. When you finish, it downloads a ZIP, and you do one push to GitHub to publish.

The id becomes the image folder name. Don't reuse: intro, tissues, skin, bone, skull, appendicular, boneaxial2.

Adding cards, or have made loops before? Drop your current custom-loops.js here so nothing you already built is lost. Brand-new and never used the builder? Skip this.

5Download

To publish: unzip it. Put the images/<id> folder into your repo's images/ (for "add" mode, the new picture files drop into that loop's existing folder), and replace custom-loops.js. Commit and push in GitHub Desktop. The card updates automatically.
